Model Context Protocol Integration

The Model Context Protocol (MCP) enables AI agents to access and process AWS cost and resource data through specialized servers.

Cost Explorer MCP Server
Provides access to detailed AWS cost data, usage patterns, and spending forecasts for optimization analysis.
CloudWatch MCP Server
Delivers resource utilization metrics to identify underutilized resources and right-sizing opportunities.
Tagging MCP Server
Provides resource tagging information for cost allocation by team, project, or environment.
