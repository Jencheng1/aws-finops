System Architecture

AWS Bedrock Foundation
Provides the AI foundation models and agent capabilities that power the entire system.
Orchestrator Agent
Coordinates all specialized agents and manages the overall optimization workflow.
MCP Servers
Model Context Protocol servers provide agents with access to AWS cost data, metrics, and tagging information.
Agent-to-Agent Communication
Enables specialized agents to collaborate and share insights for comprehensive optimization.
Streamlit Frontend
Interactive dashboard for visualizing costs, recommendations, and optimization metrics.
External Integrations
Connects with third-party FinOps tools like CloudHealth, Cloudability, and AWS Trusted Advisor.
