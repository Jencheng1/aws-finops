Implementation & Demo

1
Set Up AWS Bedrock Agents
Configure specialized agents for EC2, S3, and RDS services.
# Create Bedrock agent
aws bedrock-agent create-agent \
  --agent-name "FinOps-EC2-Agent" \
  --foundation-model "anthropic.claude-v2"
2
Deploy MCP Servers
Deploy Cost Explorer, CloudWatch, and Tagging MCP servers.
# Deploy MCP server
sam deploy --template-file mcp-server.yaml \
  --stack-name cost-explorer-mcp
3
Launch Streamlit Dashboard
Deploy the Streamlit frontend application for visualization.
# Run Streamlit app
cd finops-copilot/frontend
streamlit run app.py
4
Integrate External Tools
Connect with CloudHealth, Cloudability, and AWS Trusted Advisor.
